# Advanced bioinformatics course on pathogen genomics - FA5 2026

This repository contains the materials for the advanced bioinformatics course on pathogen genomics, created by a collaboration between researchers from the [Armauer Hansen Research Institute](https://ahri.gov.et/), the [Institute of Tropical Medicine Antwerp](https://www.itg.be/en/) and the [University of Antwerp](https://www.uantwerpen.be/en/), and supported by the [FA5 programme](https://www.itg.be/en/study/collaboration).

This is a continuation of the [FA5 2025 introductory course](https://cuypers-wim.github.io/FA5-bioinformatics/) and continues in its goals of supporting the implementation of sequencing bioinformatics to strengthen molecular surveillance and pathogen research.

> The course notes for this repository are available at the following URL: [https://ahri-itm-ua.github.io/FA5-advanced-bioinformatics-course-2026/](https://ahri-itm-ua.github.io/FA5-advanced-bioinformatics-course-2026/).
